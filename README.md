# Chat Real Time Socket.Io - Node - Express - MongoDB

Simple Chat using Socket.io with docker and docker-compose

## Instalation

Install all dependencies

```
npm install
```

Run app

```
npm start
```
## For docker

Run compose

```
docker-compose up
```
## Built With

* Node.Js
* Express.Js
* MongoDB
* Socket.IO
